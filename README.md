# [미션] Spring Boot JPA로 게시판 구현

## 미션 소개 😎
Spring Boot JPA - Rest API를 강의를 듣고, 게시판 구현 미션을 수행해봅시다.

## 이곳은 공개 Repo입니다.
1. 회원(User)와 게시글(Post) 엔티티를 구성한다.
2. 회원과 게시글에 대한 연관관계를 설정한다.
3. 게시글 Repository를 구현한다.
4. API를 구현한다.
   1. 게시글조회
      1. 페이징 조회
      2. 단건 조회
   2. 게시글 작성
   3. 게시글 수정
5. REST_DOCS를 이용해서 문서화한다.
